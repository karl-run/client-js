Display the current patient

NOTE: Authenticated clients will have patient.id. This example uses an open FHIR server so we pass a patient to manually specify which the current patient is.

Code:
const client = FHIR.client({
    serverUrl: "https://r3.smarthealthit.org",
    tokenResponse: {
        patient: "2e27c71e-30c8-4ceb-8c1c-5641e066c0a4"
    }
});

const patient = await client.request(`Patient/${client.patient.id}`);
